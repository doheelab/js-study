## CSS 절대적인 유닛

- px: 모니터 위의 가장 작은 단위

- 컨테이너가 변해도 크기가 고정

- 브라우저에서 폰트 사이즈 변경 바꿔도 고정

## CSS 상대적인 유닛

- em: 부모를 기준으로 크기를 결정

  - 폰트에 상관없이 같은 사이즈

  - 부모의 폰트 사이즈에 상대적으로 지정
    - 0.5em = parent의 0.5배 = 50%
    - 4em = parent의 4배 = 400%

- rem: root를 기준으로 크기를 결정

- vw, vh: 보이는 크기에 따라 지정

- %: 부모를 기준으로 크기를 결정
