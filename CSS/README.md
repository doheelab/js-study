# padding, margin, border

> padding

- 컨텐츠 안에 들어가는 space

- padding-top, padding-bottom, padding-left, paddin-right 작성 가능

- padding: 10px 10px 10px 10px

- padding: 20px 0px

> margin

- margin: 컨텐츠 밖에 들어가는 space

> border

- border-width, border-style, border-color 설정 가능

- border: 2px dashed red;
